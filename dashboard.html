How to Add New Products

1️⃣ Go to your GitHub repo → images/ → Upload photo

2️⃣ Copy the image name (example: photo1.png)

3️⃣ Go to products.json → Add like this:

[
  {
    "name": "Smart Watch",
    "price": 999,
    "image": "watch1.png"
  }
]

4️⃣ Save → Website auto update.
